=== ХМАО ВОИ — Инструкция по установке ===

Версия темы: 1.0.0
Требуется WordPress: 6.0+
Требуется PHP: 8.0+

--- УСТАНОВКА ---

1. Войдите в панель управления WordPress (ваш-сайт.ru/wp-admin)
2. Внешний вид → Темы → Добавить тему → Загрузить тему
3. Выберите файл hmao-voi.zip
4. Нажмите «Установить»
5. Нажмите «Активировать»

ВСЁ! При активации тема автоматически:
 - Создаст все 10 страниц (О нас, Структура, Конвенция и т.д.)
 - Создаст и настроит главное меню
 - Подключит все кастомные типы записей

--- УПРАВЛЕНИЕ КОНТЕНТОМ ---

НОВОСТИ → Записи → Добавить
ПРОЕКТЫ → Проекты → Добавить проект
МЕРОПРИЯТИЯ → Мероприятия → Добавить мероприятие
ДОКУМЕНТЫ → Документы → Добавить документ
КОМАНДА → Сотрудники → Добавить сотрудника
ВЕЛИКИЕ ИНВАЛИДЫ → Великие инвалиды → Добавить
ФОТОГРАФИИ → Фотоальбомы → Добавить (добавляйте блок «Галерея» в редакторе)

--- КАК ЗАГРУЗИТЬ ДОКУМЕНТ ДЛЯ СКАЧИВАНИЯ ---

1. Медиафайлы → Добавить новый → загрузите PDF/DOCX/XLSX
2. Скопируйте URL загруженного файла
3. Документы → Добавить документ
4. Введите название
5. В поле «URL файла» вставьте скопированный URL (или нажмите «Выбрать файл»)
6. Укажите размер, тип, категорию
7. Опубликуйте

Кнопка «Скачать» появится автоматически и будет скачивать файл.

--- ВЕРСИЯ ДЛЯ СЛАБОВИДЯЩИХ ---

Кнопка в правом верхнем углу. Настройки (цвета, размер шрифта):
Внешний вид → Настроить → «Версия для слабовидящих»

--- ДОСТИЖЕНИЯ (главная страница) ---

Внешний вид → Настроить → «Блок Достижения»
Меняйте цифры и подписи без кода.

--- КОНТАКТЫ ---

Внешний вид → Настроить → «Контакты организации»

--- МЕНЮ ---

Внешний вид → Меню
Добавляйте/убирайте пункты стандартными средствами WordPress.

--- СОЗДАНИЕ НОВЫХ СТРАНИЦ ---

Страницы → Добавить новую
Создайте страницу, опубликуйте.
Потом добавьте её в меню: Внешний вид → Меню.

--- ЛОГОТИП ---

Внешний вид → Настроить → «Идентификация сайта» → «Логотип»
Рекомендуемый размер: 80×80 пикселей, PNG с прозрачным фоном.
