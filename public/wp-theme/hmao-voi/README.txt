=== ХМАО ВОИ — Инструкция по установке ===

Версия темы: 2.0.0 (Gutenberg + CPT)
Требуется WordPress: 6.3+
Требуется PHP: 8.0+

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
УСТАНОВКА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Войдите в wp-admin
2. Внешний вид → Темы → Добавить → Загрузить
3. Выберите hmao-voi.zip → Установить → Активировать

ГОТОВО! При активации автоматически создаются:
 ✓ 12 страниц с демо-контентом (блоки Gutenberg)
 ✓ Главное меню из 11 пунктов
 ✓ 5 сотрудников
 ✓ 3 документа
 ✓ 3 проекта
 ✓ 3 мероприятия
 ✓ 3 великих человека
 ✓ 3 новости
 ✓ 1 фотоальбом
 ✓ Таблица обращений в БД

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
КАК РЕДАКТИРОВАТЬ СТРАНИЦЫ (ГЛАВНОЕ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Страницы → Главная → нажмите «Редактировать»
Вы увидите блоки Gutenberg. Каждый блок можно:
 - Кликнуть и изменить текст прямо в редакторе
 - Перетащить в другое место (иконка :: слева)
 - Удалить (три точки → Удалить блок)
 - Добавить новый (кнопка + в редакторе)

КАСТОМНЫЕ БЛОКИ (в разделе «ХМАО ВОИ»):
 • Герой — баннер с заголовком и кнопкой
 • Текстовый раздел — текст с меткой
 • Цифры и факты — карточки со статистикой
 • Достижения — карточки с иконками и описанием
 • Контакты — адрес, телефон, email, соцсети
 • Карта — Google Maps (укажите координаты)
 • Форма обратной связи — AJAX-форма

УПРАВЛЕНИЕ КОНТЕНТОМ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

НОВОСТИ        → Записи → Добавить запись
ПРОЕКТЫ        → Проекты → Добавить проект
МЕРОПРИЯТИЯ   → Мероприятия → Добавить мероприятие
ДОКУМЕНТЫ      → Документы → Добавить документ
КОМАНДА        → Команда → Добавить сотрудника
ВЕЛИКИЕ ЛЮДИ  → Великие люди → Добавить персону
ФОТОАЛЬБОМЫ   → Фотоальбомы → Добавить (используйте блок «Галерея»)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
КАК ЗАГРУЗИТЬ ДОКУМЕНТ ДЛЯ СКАЧИВАНИЯ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Медиафайлы → Добавить → загрузите PDF/DOCX/XLSX
2. Документы → Добавить документ
3. Введите название
4. В боковой панели нажмите «Выбрать файл» — откроется медиатека
5. Укажите размер, тип (PDF/DOCX/…), категорию
6. Опубликуйте — кнопка «Скачать» появится автоматически

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ФОРМА ОБРАТНОЙ СВЯЗИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Работает сразу. Обращения сохраняются:
wp-admin → «Обращения» (отдельный пункт в меню)
Копия приходит на email администратора сайта.

ВЕРСИЯ ДЛЯ СЛАБОВИДЯЩИХ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Кнопка-глаз в шапке. Режим запоминается в браузере.

ЛОГОТИП
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Внешний вид → Настроить → Идентификация сайта → Логотип
Рекомендованный размер: 80×80 пикселей, PNG с прозрачным фоном.
